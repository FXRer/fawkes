ffwiviewer(1)
=============

NAME
----
ffwiviewer - Fawkes world info viewer

SYNOPSIS
--------
[verse]
*ffwiviewer*

DESCRIPTION
-----------
This GUI application listens on the network to world info protocol
multicast communication and draws the received information on the
screen. A soccer scenario is assumed.
