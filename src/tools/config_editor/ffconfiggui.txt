ffconfiggui(1)
==============

NAME
----
ffconfiggui - View and edit a Fawkes configuration remotely

SYNOPSIS
--------
[verse]
*ffconfiggui*

DESCRIPTION
-----------
This GUI application allows for viewing and editing the configuration
of a running Fawkes instance remotely.
